Astrometry.net (craftutils.wrap.astrometry_net)
===============================================

Astrometry.net is a code for guessing the astrometric solution of an image.
It is relatively fast and can be quite precise when provided with good catalogues.
This module provides utilities for generating the files Astrometry.net requires to run and for running it.
